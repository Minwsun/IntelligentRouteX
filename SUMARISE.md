# Legacy Handoff Note

`SUMARISE.md` không còn là file canonical của repo.

Từ bây giờ, hãy dùng các file chính thức dưới `docs/`:

- `docs/idea/idea.md`
- `docs/architecture/architecture.md`
- `docs/summarize/summarize.md`
- `docs/result/result-latest.md`

Nếu cần lịch sử cập nhật, dùng các thư mục:

- `docs/idea/history/`
- `docs/architecture/history/`
- `docs/summarize/history/`
- `docs/result/history/`

Ghi chú:

- nội dung handoff cũ vẫn còn trong git history
- file này được giữ lại như một điểm điều hướng để tránh hai nguồn canonical song song
